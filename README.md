# AI Task Portal

A lightweight, single-file task tracker. No build tools or dependencies — just open `index.html` in a browser.

## Features

- Add tasks with a priority (low/medium/high) and optional due date
- Mark tasks complete, delete tasks, clear completed
- Filter by all / active / completed
- Overdue tasks are flagged automatically
- Tasks persist locally via `localStorage`

## Usage

Open [index.html](index.html) directly in any modern browser. No server or install step required.
